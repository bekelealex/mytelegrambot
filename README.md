# 🤖 English Mastery Telegram Bot

[![Deploy with GitHub Actions](https://img.shields.io/badge/Deploy-GitHub%20Actions-blue)](https://github.com/features/actions)

##  Features
- 100 advanced English questions
- Grammar mastery (tenses, conditionals, modals)
- Detailed explanations
- Progress tracking

##  Deployment
This bot runs on GitHub Actions - completely free!

##  Setup
1. Add `BOT_TOKEN` to GitHub Secrets
2. Push to main branch
3. Bot automatically starts!

##  Quiz Categories
- Reading Passage Completion
- Modal Auxiliaries  
- All Conditionals
- Reported Speech
- Advanced Vocabulary
- Reading Comprehension
- All Tenses Mastery
